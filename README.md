# PayrollSystem

Ensure g++ compiler is installed first.

To compile:

1. Open command prompt/Open terminal(for VS Code)
2. Run the following line: g++ -std=c++11 -c Pay-Sys.cpp -o Pay-Sys.o
3. Run the following line: g++ -o Pay-Sys.exe Pay-Sys.o

To run:
